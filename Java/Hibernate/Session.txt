====== Session ======
Created Четверг 27 Апрель 2017

Сессия используется для получения физического подключения к базе.

Так как объект сессии не потоко безопасен, поэтому они не должны существовать долгое время. You should always use "one session per request" or "one session per transaction.

Transient - новый экземлпяр класса не связанный с сессией и не имеющий отражения в базе

Persistent - экземпляр, связанный с сессией. Имеет отражение в базе

Detached - persistent экземпляр после того как мы закрыли сессию
