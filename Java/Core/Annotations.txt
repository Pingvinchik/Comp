[[RetentionPolicy]]

Source - аннотация видна только в source, в байт коде ее не будет, пример @Overrride 
Class - должна попасть в байт код, но ее невозможно будет считать в runtime через reflection
Runtime - видный в runtime и можно считать через reflection
