====== Core ======
Created Четверг 06 Апрель 2017

jvm - Java Virtual Machine - среда ("виртуальная"), в которой выполняется твой скомпилированный код
jre - Java Runtime Environment. Просто пакет, который включает в себя JVM и минимальный набор библиотек для работы программ. И еще браузерный плагин, где будут выполняться апплеты.
jdk - Java Development Kit - это уже полноценный набор библиотек и инструментов для для создания, компилирования и дебага программ. Включает в себя jre

JRE - джава рантайм энвайермент, на пальцах если, то это среда для запуска джава программ (уже скомпелированных классов, а не исходников или джар-файлов) 
JDK - джава девелопмент кит, опять на пальцах, инструмент для создания (сборки) джава программ. в составе имеется само собой компиллятор и вышеупомянутая JRE 	

JRE -- программный комплекс, позволяющий исполнять программы для JVM, написанные, к примеру, на языке java. JVM (java virtual machine) идет в комплекте :) 
JDK -- программный комплекс, позволяющий Вам самим создавать программы для JVM на языке java
