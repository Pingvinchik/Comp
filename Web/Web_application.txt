====== Web application ======
Created Суббота 13 Май 2017

Servlet container - отвечает за жизненный цикл сервлетов (создание, вызов, уничтожение)
Servlet context (root context) - хранит конфигурационные данные (различные параметры, настройки приложения)

Filter - перед тем как запрос клиента попадет в java код проходит фильтры
